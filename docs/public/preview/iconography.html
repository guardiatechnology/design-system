Como importar no projeto React + Vite/Next

1. Instalar

lucide-react — pacote oficial, tree-shakable. Cada ícone é um componente React.

# npm npm install lucide-react # pnpm / yarn pnpm add lucide-react yarn add lucide-react

2. Importar e usar

Importe apenas os ícones que precisar — o bundler remove o resto.

import { Search, Check, CircleCheck } from "lucide-react"; export function Example() { return ( <button> <Search size={16} /> Buscar </button> ); }

3. Nomes kebab-case → PascalCase

Na referência da Guardia os nomes são kebab-case (convenção Lucide). Em código React, use PascalCase:

// referência do DS "arrow-left-right" → ArrowLeftRight "book-open-check" → BookOpenCheck "circle-check" → CircleCheck "trending-up" → TrendingUp

4. Cor e tamanho

Ícones Lucide usam stroke="currentColor" — herdam a cor do texto. Nunca defina cor inline.

// ✓ bom — herda a cor do botão <button className="btn-primary"> <Check size={16} /> </button> // ✗ ruim — cor cristalizada <Check color="#7140AB" />
Regras do · don't
Consistência no domínio
Use o mesmo ícone para a mesma ação em toda a UI. Se "Excluir" é trash-2 numa tela, é trash-2 em todas.
Não componha ícones
Não empilhe file + plus para criar "Novo arquivo". Use file-plus — a biblioteca já tem.
Consistência no domínio
Ações direcionais usam arrow-*; disclosure usa chevron-*. Nunca misture.
Não use como único sinal
Ícone sozinho exige aria-label ou title. Não substitua texto por ícone em CTAs críticos.
