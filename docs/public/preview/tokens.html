FUNDAMENTOS · TOKENS

Tokens do DS

Catálogo completo dos design tokens disponíveis em colors_and_type.css. Todo componente do DS consome estas variáveis — para criar um novo padrão, comece aqui. Clique em qualquer nome para copiar a chamada var(--…).

Como importar no projeto

Um único arquivo CSS expõe todos os tokens como variáveis nativas.
1. Copiar o arquivo

Copie colors_and_type.css para a raiz do seu projeto (ou onde preferir centralizar tokens). 

// estrutura sugerida
src/
├── styles/
│   └── colors_and_type.css  ← tokens
└── app.css                  ← seu código
2. Importar no CSS raiz

Use @import no topo do seu CSS principal — tokens ficam disponíveis em toda a app. 

/* app.css */
@import "./styles/colors_and_type.css";

/* agora use os tokens */
.my-card {
  background: var(--surface);
  border: 1px solid var(--border);
  border-radius: var(--radius-md);
  padding: var(--space-4);
}
3. Alternativa · link no HTML

Em páginas estáticas ou protótipos, inclua antes dos seus outros stylesheets. 

<!-- index.html -->
<link rel="stylesheet"
      href="styles/colors_and_type.css">
<link rel="stylesheet"
      href="app.css">
4. Uso em TS/JS (opcional)

Para ler tokens de runtime (ex.: Chart, Canvas), use getComputedStyle. 

// lê o valor atual do token
const violet = getComputedStyle(
  document.documentElement
).getPropertyValue("--violet-500").trim();

// "#4F186D"

Princípio nº 1 — sempre consuma var(--token) em vez de hex cru. Trocar um valor em colors_and_type.css propaga automaticamente para todos os componentes.
