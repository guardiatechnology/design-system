Lastica · Poppins · Roboto

Lastica é reservada aos logotipos oficiais. Poppins é a fonte corrente em toda comunicação. Roboto é o fallback oficial para ambientes onde Poppins não está disponível.

Lastica · EXCLUSIVA para logotipos
Guardia
Sans-serif geométrica criada por Alberto Fontense. Uso limitado à construção dos logotipos, assinatura oficial e aplicações em que a marca aparece como selo. Nunca em UI, documentos ou corpo de texto.

Arquivo oficial

TTF disponível em assets/fonts/Lastica.ttf. Uso restrito — consulte a equipe de Brand antes de incluir em qualquer peça.

@font-face

/* apenas para renderizar logos */
@font-face {
  font-family: 'Lastica';
  src: url('assets/fonts/Lastica.ttf')
       format('truetype');
  font-display: swap;
}

Google Fonts · HTML

Recomendado para site e produto. Zero peso de setup, cache CDN global.

<!-- <head> -->
<link rel="preconnect"
      href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Poppins:ital,wght@0,100..900;1,400;1,500&display=swap"
      rel="stylesheet">

Google Fonts · CSS @import

Para ambientes sem acesso ao <head>.

/* topo do seu CSS */
@import url('https://fonts.googleapis.com/css2?family=Poppins:ital,wght@0,100..900;1,400&display=swap');

body {
  font-family: 'Poppins', 'Roboto', sans-serif;
}

Self-hosted · @font-face

Para apps internas, offline, ou controle total sobre versão. TTFs em assets/fonts/poppins/.

@font-face {
  font-family: 'Poppins';
  src: url('./poppins/Poppins-Regular.ttf')
       format('truetype');
  font-weight: 400;
  font-display: swap;
}
/* repita p/ cada peso usado */

npm · @fontsource (React/Vite/Next)

Bundle otimizado por peso. Importe só os pesos que você usa.

// terminal
npm install @fontsource/poppins

// app.tsx ou layout raiz
import "@fontsource/poppins/400.css";
import "@fontsource/poppins/500.css";
import "@fontsource/poppins/600.css";
import "@fontsource/poppins/700.css";

Hierarquia idêntica à Poppins (substituição direta por peso). Uso: e-mails nativos, sistemas restritos à importação de fontes, ambientes onde Poppins não carrega.

Google Fonts · HTML

Mesma via da Poppins — adicione ao mesmo <link>.

<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;500;600;700&display=swap"
      rel="stylesheet">

Cadeia de fallback recomendada

Defina uma vez e use em todo o projeto.

/* colors_and_type.css já expõe: */
:root {
  --font-sans: 'Poppins',
                 'Roboto',
                 system-ui, sans-serif;
}

body { font-family: var(--font-sans); }
